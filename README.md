# hb_flutter_util

A common Utils for Flutter.

## Install Started

1. Add this to your **pubspec.yaml** file:

```yaml
dependencies:
  hb_flutter_util: ^0.0.1
```

2. Install it

```bash
$ flutter packages get
```

## Configuration

## add properties in info.plist in iOS

```
<key>LSApplicationQueriesSchemes</key>
<array>
    <string>wechat</string>
</array>
```
